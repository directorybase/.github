# DirectoryBase

This is the system behind the tools.

It’s where everything connects.

---

## What this is

directorybase is a set of projects built around one problem:

AI output looks right.

But it breaks when you try to use it.

Existing tools just didn't for well enough, so I built these myself.

---

## The parts

These projects work together:

### datatool.dev
Fix broken AI-generated JSON.

### toolidx.dev
Structured index of AI tools with verification.

### agenticwatch.dev
Real-world testing and notes on AI tools.

---

## How it fits together

- datatool fixes structure  
- toolidx provides structured data  
- agenticwatch tests what actually works  

Each solves a different part of the same problem.

---

## Why this exists

Most tools assume clean input.

AI doesn’t give clean input.

So things fail in small ways:

- broken JSON  
- missing fields  
- tools that don’t install  
- outputs you can’t trust  

This system is built to catch that early.

---

## Focus

- structure first  
- no guessing  
- fail safely  
- test against real examples  

---

## Notes

This is not a directory of random projects.

Everything here is connected.
